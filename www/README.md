# 📢 Islamic Dawah This Century — الدعوة الإسلامية في القرن الحالي

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **الدعوة الإسلامية في القرن الحالي** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on modern dawah methods, 5 contemporary challenges, daily habit tracker, self-quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes, floating particles, glassmorphism panels, swipe navigation
- Responsive design + PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.

## Disclaimer

I am not a scholar. This is a humble effort to share Sheikh al-Ghazali's wisdom interactively. Not a fatwa.
